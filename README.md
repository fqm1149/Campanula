# Campanula Bot
Campanula 是一个为 Generals 游戏开发的 AI 机器人（Bot），采用 C 语言编写，具备基本的游戏决策能力和路径规划功能。
## 功能特性
### 已完成功能
- 完整的通信框架：通过 TCP socket 与游戏服务器连接

- 游戏状态解析：解析地图数据、玩家信息、游戏设置

- 路径规划算法：计算点到点、点到边界、点到敌人等多种距离

- 反作弊机制：根据游戏规则限制视野范围

- 内存管理：完善的动态内存分配和释放机制

### 开发中功能
* 核心决策逻辑：基于权重计算做出移动决策

* 多种策略模式：扩张、攻击、防守等不同游戏阶段的策略

* 性能优化：算法缓存和计算优化
